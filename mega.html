Megatron

This page is about my bash bot, Megatron.

Overview

Megatron is an IRC bot. I use it on EvilCorp IRC. Megatron has been through a lot but I can not take pride in the base line to the code. I couldnt tell ya who first made the code cause its a long rabbit hole in GitHub. I just know I fell in love with it and have created Megatron. Formerley known as Rebel_Joe for my old #Rebels channel in Evilcorp. I had a corrupted hard drive and couldnt get the files back so I had to start over. I have come far with this code and you will soon be able to view its source code on GitHub. <3

Megatron Tricks

- Seen = Tells you when and what a user last sent to the channel.

- Fortune = Uses bash frtune program.

- 8ball = answer a question via yes, no, and maybe.

- Urban = Uses a custom bash script to get urban responses.

- Insult = Insults a user of ur choice.
